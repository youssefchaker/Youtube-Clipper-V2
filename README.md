# YouTube Clipper

An extension made for chromium browsers that lets you capture and download short clips from any YouTube video — directly from your browser, no external tools needed.

---

## What It Does

- Adds a **"Clip"** button on every YouTube video
- Lets you pick a **start and end time** (up to 1 minute)
- Records the video segment and saves it as a file you can share anywhere

---

## Installation

### Step 1: Download the Extension Files

1. Download the project ZIP file to your computer
2. Extract (unzip) it to a folder you can easily find

### Step 2: Enable Developer

1. Open browser
2. Type this in the address bar and press **Enter**:
   ```
   chrome://extensions/
   ```
3. Look for the **"Developer mode"** toggle
4. Turn it **ON**

### Step 3: Load the Extension

1. Click the **"Load unpacked"** button
2. A file picker will open — navigate to the folder where you extracted the ZIP
3. **Select the folder** (not individual files) and click **"Select Folder"**

---

## How to Use

### 1. Open Any YouTube Video

### 2. Click the "Clip" Button

### 3. Set Your Start and End Times

* Play the video and pause at the moment you want the clip to begin

* Click "Set Start" — the start time fields update automatically

* Play the video and pause at the moment you want the clip to end

* Click "Set End" — the end time fields update automatically

Tip: You can also type times manually in the HH:MM:SS fields if you know the exact timestamps.
