# YouTube Clipper

A Chrome extension that lets you capture and download short clips from any YouTube video — directly from your browser, no external tools needed.

---

## What It Does

- Adds a **"Clip"** button on every YouTube video
- Lets you pick a **start and end time** (up to 1 minute)
- Records the video segment and saves it as a file you can share anywhere

---

## Installation

### Step 1: Download the Extension Files

1. Download the project ZIP file to your computer
2. Extract (unzip) it to a folder you can easily find

### Step 2: Enable Developer Mode in Chrome

1. Open Google Chrome
2. Type this in the address bar and press **Enter**:
   ```
   chrome://extensions/
   ```
3. Look for the **"Developer mode"** toggle in the top-right corner
4. Turn it **ON** (it will turn blue)

### Step 3: Load the Extension

1. Click the **"Load unpacked"** button that appears in the top-left
2. A file picker will open — navigate to the folder where you extracted the ZIP
3. **Select the folder** (not individual files) and click **"Select Folder"**
